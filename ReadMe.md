# Conceptly 

Conceptly is an AI-powered browser extension that detects complex technical terms and mathematical equations on any webpage and provides instant, interactive breakdowns.

## Features
- ELI5 explanations
- Real-world analogies
- Step-by-step breakdowns
- Code examples
- Math equation rendering
- Difficulty toggle (Beginner / Intermediate / Advanced)
- Learning history tracking

## Tech Stack
- Chrome Extension (Manifest V3)
- HTML, CSS, JavaScript
- AI API Integration
- MathJax / KaTeX
- Chrome Local Storage

## Problem Solved
Students often struggle with complex terminology and need to switch tabs frequently to understand concepts. Conceptly eliminates context switching and improves conceptual clarity.
